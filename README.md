# My Portfolio

A simple personal portfolio website built as a single-page layout to showcase my introduction, skills, projects, and contact links.

## Overview

This project is designed to present work in a clean and organized way with a responsive layout that works across different screen sizes. It uses:

- HTML for structuring the content and sections.
- CSS for styling the interface and giving it a modern look.
- JavaScript for adding a simple reveal effect while scrolling.

## File Structure

```text
My_portfolio/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Page Content

- A hero section for a quick introduction.
- A short personal bio.
- A technical skills section.
- A featured projects section.
- Social and contact links.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## How to Run

- https://abdullah-portfolio-3pb8.onrender.com

## Note

The page is built with a right-to-left layout for Arabic content and is structured to work well as a portfolio presentation file.
